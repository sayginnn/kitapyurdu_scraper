 kitapyurdu_scraper
* Python ve BeautifulSoup kullanarak Kitapyurdu çok satanlar listesini çeken ve analiz eden veri madenciliği projesi.
=======================================================================

Kitapyurdu Best Sellers Scraper
* Bu proje, Python kullanarak Türkiye'nin popüler kitap satış sitesi Kitapyurdu.com üzerinden "Haftalık Çok Satanlar" listesindeki ilk 10 kitabın verisini otomatik olarak çeken bir veri madenciliği (web scraping) uygulamasıdır.
=======================================================================

Projenin Amacı
* Endüstri Mühendisliği bakış açısıyla, manuel veri toplama süreçlerini otomatize ederek piyasa analizi ve fiyat takibi için ham veri üretmek.
  
=======================================================================

Kullanılan Teknolojiler
- Python 3
- - BeautifulSoup4: HTML/CSS ayrıştırma ve veri çekme.
- Urllib: HTPP istek yönetimi
 =======================================================================

Teknik Özellikler
- Tam Maskeleme (User-Agent Spoofing): Site engellemelerini aşmak için gerçek bir tarayıcı (Chrome) simülasyonu.
- CSS Selectors: Veri çekme işleminde nokta atışı doğruluk için 'select' metodu kullanımı.
- Veri Tekilleştirme (Deduplication): Sitenin HTML yapısından kaynaklanan çift kayıtların algoritmik olarak temizlenmesi.
- Hata Yakalama (Error Handling): Bağlantı sorunlarına karşı 'try-except' blokları ile koruma.
=======================================================================

Geliştirme Notu
  * Bu proje, kod verimliliğini artırmak ve modern yazılım araçlarını deneyimlemek amacıyla **Yapay Zeka (AI) asistanı desteğiyle** geliştirilmiştir.
